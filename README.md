## 🎮 **How to Play?**  
1️⃣ Choose one: **Rock (🪨), Paper (🖐️), or Scissors (✂️)**.  
2️⃣ The computer randomly selects its choice.  
3️⃣ The winner is determined based on these rules:  
   - **Rock beats Scissors**  
   - **Scissors beats Paper**  
   - **Paper beats Rock**  
4️⃣ If both choices are the same, it’s a **draw**!
